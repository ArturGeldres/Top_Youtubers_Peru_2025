## 🎯 DESCRIPCIÓN DEL PROYECTO

> *“No se trata de quién tiene más suscriptores, sino de quién genera un mejor retorno de inversión (ROI).”*

Este proyecto nace de una necesidad crítica en el mercado peruano: **identificar a los creadores de contenido más rentables**. A través de un flujo **End-to-End**, extraje datos de la API de YouTube, garanticé la calidad mediante SQL, y creé un modelo de rentabilidad visualizado en Power BI y validado en Excel.

## 💡 INSIGHTS CLAVE (Data-Driven)

Basados en el análisis profundo del top 100 de YouTubers en Perú:

- **💎 Líderes de Rentabilidad:** Canales como **RoxiCake Gamer** y **Drawblogs** presentan el mayor potencial de utilidad neta, superando los _**$400,000**_ de ganancia proyectada tras descontar los costos de campaña.
- **📊 Eficiencia de Contenido:** El promedio de vistas por video es un indicador _mucho más preciso_ para el **ROI** que el conteo total de suscriptores. _"Audiencia pequeña pero activa > Audiencia masiva pero pasiva"_.
- **✅ Validación de Datos:** Se logró una **consistencia del 100%** (diferencia 0) entre los cálculos de SQL y Excel, garantizando la integridad total para la toma de decisiones gerenciales.

## 🧠 CONCLUSIONES DEL ANÁLISIS

1.  **Viabilidad de Campaña:** Con un presupuesto de _**$50,000**_, la inversión es altamente viable para los primeros 5 canales; el punto de equilibrio se alcanza rápidamente por el volumen masivo de impresiones.
2.  **Sensibilidad de Conversión:** El modelo asume un _**2%**_. Si esta tasa cayera al **0.5%**, canales con menor promedio de vistas dejarían de ser rentables para este nivel de inversión.
3.  **Dominio del Nicho:** El sector de _**entretenimiento y gaming**_ se mantiene como el más atractivo para alcance masivo en el mercado peruano hacia el 2025.

---

## 🚀 RECOMENDACIONES ESTRATÉGICAS

- **🛡️ Diversificación:** No apostar los $50,000 en un solo canal. Se recomienda un **"mix" de 3 canales** para mitigar riesgos.
- **🛍️ Enfoque de Producto:** Dado que el costo del producto es de _**$5**_, la estrategia de marketing debe centrarse en **"compras por impulso"** para mantener la tasa de conversión.
- **🔄 Actualización:** Automatizar la ejecución del script de Python mensualmente para ajustar las proyecciones a la tendencia real de crecimiento.
